Stage Time Toolkit（舞台時間 統合管理ツール）
舞台の稽古や本番の進行をサポートするために作られた、ブラウザ上で動作する2in1のタイムマネジメントツールです。
アプリのインストールは不要で、スマホやタブレット、PCのブラウザからアクセスするだけですぐに使用できます。

🌟 本ツールの特徴（現場に強い安心設計）
舞台現場特有のトラブルやニーズに対応するため、以下の強力な機能を備えています。

絶対に止まらない・消えない設計
ブラウザのタブを間違えて閉じたり、スマホがスリープしてしまったりしても大丈夫です。裏側では常に「現実の時刻」を基準に計算しているため、ページを再読み込みすれば1秒の狂いもなくタイマーが復帰します。

画面消灯防止機能（WakeLock API）
タイマー稼働中は、スマホやタブレットの画面が勝手に暗くなるのを防ぎます。タブを切り替えて解除されてしまっても、戻ってきた瞬間に自動で再点灯するオートリカバリー機能を搭載しています。

1画面で2つのツールを瞬時に切り替え
「記録を取る」ためのストップウォッチと、「全体の流れを把握する」ためのバータイマーをタブで統合。どちらを開いていても、裏側でもう一つのタイマーは動き続けます。また、最後に開いていたタブを記憶し、次回アクセス時に自動展開します。

⏱️ ツール①：記録ストップウォッチ
稽古のタイム計測や、本番中のキッカケ（音響・照明・役者の動きなど）の記録に特化したストップウォッチです。

起点時刻の変更（押し・巻き対応）
スタート後に「起点時刻」を編集可能です。「開演が5分押したから、タイマーの基準も5分ずらしたい」といった現場での急な変更にも即座に対応し、全体のタイムを連動してシフトさせます。（マイナス時間からのカウントダウンスタートも可能です）

ラップ＆即時メモ機能
「Lap / Memo」ボタンを押すと、その瞬間の「経過時間」と「現実時刻」がリスト化されます。横の入力欄にメモ（例：「M1開始」「暗転」）を打ち込むと、入力した瞬間に自動保存されます。

CSVデータ出力機能
記録したラップタイムとメモは、ボタン一つでCSVファイルとしてダウンロード可能。Excel等で開き、そのまま各セクションへの記録報告として共有できます。（文字化け対策済み）

📊 ツール②：進行バータイマー
当日のタイムスケジュールや、香盤表に沿った進行状況を視覚的に把握するためのバータイマーです。

スケジュールのプログレス表示
「13:00」「14:30」「15:00」のように複数の区切り時間を設定すると、現在どの区間にいて、次の予定まであと何分何秒あるかをプログレスバーと巨大なカウントダウンで表示します。

日またぎ（24時間オーバー）対応
「23:00」から「01:00」といった、深夜をまたぐ深夜仕込み・バラシなどのスケジュール設定にも完全対応しています。

💡 動作環境
モダンブラウザ（Google Chrome, Safari, Edgeなど）の最新版が動作するスマートフォン、タブレット、PC。
※画面消灯防止機能は、お使いの端末・ブラウザの仕様により制限される場合があります（緑色・赤色のランプが点灯していれば有効です）。
